# 安装

执行命令

```
npm install https://h5-fc-code.oss-cn-hangzhou.aliyuncs.com/package/dbfront-tools/dbfront-tools-0.0.1.tgz -S
```

# 开发

### 1、目录结构描述

```text
src
├── index.ts // 组件库源码的入口文件
├── packageA
        ├── index.ts      // 组件 A 入口文件
        ├── index.md      // 组件 A 使用文档
        ├── PackageA.tsx  // 组件 A 文件
```

### 2、开发完成

推送到远程，使用 Package Deploy 将压缩包推送到 OSS
